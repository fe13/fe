# 调度 Schedule

有时我们不希望马上执行函数，而且等待一段时间再执行。一般有以下两种方式：
* `setTimeout()` 方法在在指定时间后执行某个函数。

* `setInterval()` 方法以指定的时间间隔重复执行某个函数。
