# Remove Duplicates from Sorted Array II

## JavaScript
```javascript
function removeDuplicates(nums) {
  for (let i = 0; i + 2 < nums.length; ) {
    nums[i] === nums[i + 1] && nums[i + 1] === nums[i + 2] ? nums.splice(i, 1) : i += 1
  }
  return nums.length
}
```
