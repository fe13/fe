# Array Partition I
`简单`

## JavaScript
```javascript
function arrayPairSum(nums) {
  return nums.sort((a, b) => a - b).filter((v, i) => i % 2 === 0).reduce((a, b) => a + b)
}
```

## Python
```python
class Solution(object):
    def arrayPairSum(self, nums):
        return sum(sorted(nums)[::2])
```
