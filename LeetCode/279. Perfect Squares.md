# Perfect Squares
`中等`

```javascript
function numSquares(n) {
  let results = new Array(n + 1).fill(Infinity);
  for (let i = 0; i * i <= n; i++) {
    results[i * i] = 1;
  }
  
  for (let i = 0; i <= n; i++) {
    for (let j = 1; j * j <= i; j++) {
      results[i] = Math.min(results[i], results[i - j * j] + 1);
    }
  }

  return results[n];
};
```
