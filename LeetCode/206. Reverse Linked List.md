# Reverse Linked List
`简单`

## JavaScrit
```javascript
function reverseList(head) {
  let curr = head, prev = null
  while (curr) {
    let next = curr.next
    curr.next = prev
    prev = curr
    curr = next
  }
  return prev
}
```
递归解法
```javascript
function reverseList(head) {
  if (head === null || head.next === null) return head
  let curr = head, next = head.next
  let list = reverseList(next)
  next.next = head
  head.next = null
  return list
}
```
