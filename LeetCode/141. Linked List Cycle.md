# Linked List Cycle
`简单`

## JavaScript
```javascript
function hasCycle(head) {
  if (head === null) return false
  let walker = head
  let runner = head
  while (walker.next && runner.next && runner.next.next) {
    walker = walker.next
    runner = runner.next.next
    if (walker === runner) return true
  }
  return false
}
```
