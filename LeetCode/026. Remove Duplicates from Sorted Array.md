# Remove Duplicates from Sorted Array
`简单`

注意多个重复的情况 `[1, 1, 1]`

## JavaScript
```javascript
function removeDuplicates(nums) {
  for (let i = 0; i < nums.length; i++) {
    if (nums[i] === nums[i + 1]) {
      nums.splice(i, 1)
      i -= 1
    }
  }
  return nums.length
}
```
